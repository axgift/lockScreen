# 时间管理屏幕保护程序

一个基于Python的桌面应用程序，用于管理和限制电脑使用时间。

## 快速开始

### 方法1：使用批处理文件（推荐）
- 双击 `启动程序.bat` 启动程序
- 双击 `停止程序.bat` 停止程序
- 双击 `重置配置.bat` 重置所有设置（忘记密码时使用）

### 方法2：使用命令行
```bash
# 启动程序
py main.py

# 停止程序
taskkill /F /IM python.exe
```

## 功能特性

### 核心功能
- 📅 **灵活的时间设置**：为周一至周日分别设置可用时间段
- 🔒 **全屏锁定**：不可用时间自动全屏锁定电脑
- 🔐 **双重密码保护**：管理员密码和临时退出密码
- ⏰ **实时监控**：后台自动检查时间并执行锁定

### 增强功能
- 🎯 **系统托盘**：最小化到系统托盘，支持快速操作
- 🚀 **开机自启动**：随Windows自动启动
- ⏳ **临时解锁**：可设置临时解锁时间限制
- ⚠️ **锁定前提醒**：锁定前弹出警告
- 🔑 **密码修改验证**：修改密码需要验证旧密码
- 🆘 **紧急解锁**：安全问题/答案验证解锁
- 📊 **使用统计**：记录每日使用时间
- 🛡️ **进程保护**：防止程序被意外关闭
- ✅ **白名单应用**：锁定时间仍可使用指定应用

## 使用指南

### 首次设置
1. 运行程序，进入"密码设置"标签
2. 设置管理员密码（必须）
3. 设置临时退出密码（可选）
4. 设置安全问题和答案（紧急解锁使用）

### 时间设置
1. 进入"时间设置"标签
2. 为每天设置可用时间段
3. 点击"保存时间设置"

### 系统设置
1. 进入"系统设置"标签
2. 设置开机自启动
3. 配置临时解锁持续时间
4. 配置锁定前提醒时间

### 白名单应用
1. 进入"白名单应用"标签
2. 点击"扫描运行中的应用"
3. 选择需要白名单的应用
4. 点击"添加选中"

### 查看统计
1. 进入"使用统计"标签
2. 查看每日使用时间
3. 可清除历史统计数据

## 快捷键功能

### 默认快捷键
- `Ctrl+Alt+A` - 显示/打开设置窗口
- `Ctrl+Alt+H` - 隐藏窗口
- `Ctrl+Alt+L` - 立即锁定屏幕
- `Ctrl+Alt+E` - 退出程序（需要密码）

### 自定义快捷键
1. 进入"系统设置"标签
2. 在"快捷键设置"区域修改快捷键
3. 点击"保存系统设置"
4. 快捷键会自动重新加载

**快捷键格式**：`ctrl+alt+字母`（例如：`ctrl+alt+a`）

### 快捷键使用技巧
- 快捷键全局有效，即使窗口隐藏也能使用
- 修改快捷键后需要保存才能生效
- 快捷键不区分大小写

## 系统托盘功能

右键点击系统托盘图标可以访问：

- **打开设置** - 重新显示主窗口
- **隐藏窗口** - 隐藏主窗口
- **查看状态** - 查看当前锁定状态
- **立即锁定** - 手动触发屏幕锁定
- **退出程序** - 安全退出程序（需要密码）

## 锁定与解锁

### 自动锁定
- 当当前时间不在设置的可用范围内时自动锁定

### 解锁方式
1. **临时密码解锁** - 使用临时密码解锁，有时间限制
2. **管理员密码解锁** - 使用管理员密码解锁
3. **紧急解锁** - 回答安全问题解锁
4. **自动解锁** - 时间到达可用时间段时自动解锁

## 密码管理

### 管理员密码
- 用于修改所有设置
- 用于退出程序
- 用于完全解锁

### 临时密码
- 用于临时解锁
- 可设置使用时间限制
- 适合临时使用

### 紧急解锁
- 安全问题和答案验证
- 防止忘记密码时无法访问

## 故障排除

### 程序无法启动
- 确保Python已正确安装
- 检查是否已运行程序实例
- 尝试删除 `.lock` 文件

### 无法退出程序
- 使用 `停止程序.bat`
- 或运行命令：`taskkill /F /IM python.exe`

### 忘记密码
- 使用 `重置配置.bat` 重置所有设置
- 或删除 `config.json` 文件

### 程序意外关闭
- 检查是否有杀毒软件拦截
- 确保Python环境完整

## 配置文件

程序会生成以下文件：
- `config.json` - 存储所有设置和密码（加密）
- `.lock` - 防止多实例运行
- `使用统计` - 存储在config.json中

## 安全说明

- 所有密码都使用SHA-256哈希存储
- 配置文件不包含明文密码
- 建议定期备份config.json
