## 核心模块说明

### 1. 配置管理 (`config_manager.py`)
- 采用 AES-256 加密算法保护配置文件
- 使用 PBKDF2 密钥派生函数增强安全性
- 支持机器绑定的加密密钥，防止配置文件泄露
- 单例模式确保全局配置一致性

### 2. 主窗口 (`main_window.py`)
- 整合所有子模块的核心控制器
- 提供五个功能标签页：时间设置、密码设置、使用统计、白名单应用、系统设置
- 协调时间检查、锁定控制、快捷键响应等核心流程

### 3. 时间检查器 (`time_checker.py`)
- 判断当前时间是否在可用时间段内
- 支持跨天时间段设置（如 23:00 - 02:00）
- 计算下次解锁时间和下次锁定时间

### 4. 锁定屏幕 (`lock_screen.py`)
- 全屏锁定界面，防止未授权访问
- 支持临时解锁、管理员解锁、紧急解锁三种方式
- 实时显示当前时间和下次可用时间

### 5. 快捷键管理器 (`hotkey_manager.py`)
- 全局快捷键监听
- 支持自定义快捷键配置
- 提供强制退出快捷键（Ctrl+Alt+Shift+Q）

### 6. 系统托盘 (`system_tray.py`)
- 后台运行时的快捷操作入口
- 双击图标打开设置（需验证密码）
- 提供查看状态、立即锁定、退出程序等菜单项

## 功能特性

### 时间管理
- 📅 为周一至周日分别设置独立的可用时间段
- ⏰ 实时监控当前时间，自动执行锁定/解锁
- ⚠️ 锁定前自定义时间提醒（默认 5 分钟）
- 🔄 支持跨天时间段设置

### 安全保护
- 🔐 **双重密码**：管理员密码 + 临时退出密码
- 🆘 **紧急解锁**：安全问题/答案验证机制
- 🔒 **密码加密**：采用 AES-256 算法存储密码
- 🛡️ **进程保护**：防止程序被意外关闭

### 用户体验
- 🖥️ **全屏锁定**：锁定时覆盖整个屏幕
- 📊 **使用统计**：记录每日电脑使用时间
- ✅ **白名单应用**：锁定时间内可使用指定应用
- 🚀 **开机自启**：随系统自动启动

### 便捷操作
- ⌨️ **全局快捷键**：无需打开窗口即可操作
- 📱 **系统托盘**：右键菜单快速访问功能

## 快捷键功能

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Alt+A` | 显示/打开设置窗口 |
| `Ctrl+Alt+H` | 隐藏窗口 |
| `Ctrl+Alt+L` | 立即锁定屏幕 |
| `Ctrl+Alt+E` | 退出程序（需密码） |
| `Ctrl+Alt+Shift+Q` | 强制退出 |

## 解锁方式

1. **临时密码解锁**：使用临时密码，解锁时间受限制（默认 30 分钟）
2. **管理员密码解锁**：完全解锁，可进行所有设置操作
3. **紧急解锁**：回答安全问题解锁（忘记密码时使用）
4. **自动解锁**：时间到达可用时间段时自动解锁

## 使用指南

### 首次运行

1. 启动程序后，系统会提示设置初始密码
2. 设置管理员密码（必须）和临时退出密码（必须）
3. 设置安全问题和答案（用于紧急解锁）

### 时间设置

1. 进入「时间设置」标签页
2. 为每一天设置可用时间段（可添加多个时间段）
3. 点击「保存时间设置」

### 系统设置

1. 进入「系统设置」标签页
2. 配置开机自启动、临时解锁时长、锁定前提醒时间
3. 自定义快捷键（格式：`ctrl+alt+字母`）
4. 点击「保存系统设置」

### 白名单应用

1. 进入「白名单应用」标签页
2. 在输入框中输入应用程序名（如 `notepad.exe`）
3. 或点击「浏览...」选择可执行文件
4. 点击「添加」将应用加入白名单

### 查看统计

1. 进入「使用统计」标签页
2. 查看每日使用时间记录
3. 可点击「清除统计」清空所有记录

## 安全说明

- 🔒 所有密码使用 AES-256 算法加密存储
- 📁 配置文件存储在用户数据目录（Windows：`%APPDATA%\TimeManager\TimeManager\st.pt`）
- 🔑 加密密钥绑定到用户账户和机器标识，提高安全性
- ⚠️ 建议定期备份配置文件

## 故障排除

### 程序无法启动
- 检查 Python 版本（需 3.8+）
- 确保所有依赖已正确安装
- 检查是否有其他程序实例正在运行

### 无法退出程序
- 使用快捷键 `Ctrl+Alt+Shift+Q` 强制退出

### 忘记密码
- 删除配置文件后重新运行程序（将丢失所有设置）
- 配置文件路径：`%APPDATA%\TimeManager\TimeManager\st.pt`
